                NLP#

                NLP 的范式演进历程大体经历了这样四个阶段：

                特征工程 → 深度学习 → 预训练+精调 → Prompt

                • 预训练+精调范式：让预训练模型（PLM）去适应下游任务
                • Prompt 范式：让下游任务适应预训练模型（PLM）
                  • Prompt：将下游任务的输入输出形式改造成预训练任务中的形式，即 MLM (Masked Language Model) 的形式
